# Python图书馆借阅系统

本项目是基于 Django 框架的图书管理系统，旨在提供一套便捷、高效的图书馆借阅解决方案。

## 技术栈

- **后端语言**：Python 3.11
- **前端技术**：HTML、CSS、JavaScript、Layui、ECharts
- **开发框架**：Django
- **开发工具**：PyCharm
- **数据库**：Sqlite/Mysql
- **数据库软件**：Navicat/DBeaver

## 功能模块

### 客户端功能

- 用户登录、注册、修改信息、登出
- 图书增删改查
- 数据可视化
- 图书馆公告栏展示
- 图书借阅和归还操作

### 管理页功能

- 用户管理
- 图书管理
- 借阅管理
- 公告栏管理

## 系统角色

- 普通用户：可登录、查看图书、借阅图书、还书等
- 管理员：管理用户信息、图书信息、借阅记录及公告发布

## 运行环境

- 推荐使用 Python 3.11
- 支持多种数据库环境

## 部署步骤

具体部署教程包含在项目文档中，方便用户按照步骤进行安装和配置。
